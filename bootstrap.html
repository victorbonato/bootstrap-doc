Bootstrap, o que é?

Bootstrap é um framework de código aberto bastante popular usado para criar interfaces web responsivas e visualmente atraentes. Desenvolvido originalmente para o Twitter, ele oferece uma variedade de estilos CSS, componentes JavaScript e um sistema de grid adaptável. Com o Bootstrap, é mais fácil e eficiente criar sites e aplicativos que se ajustam automaticamente a diferentes dispositivos e tamanhos de tela.

Vantagens

  1. Evita trabalhos repetitivos em projetos.
  2. Design responsivo capaz de se adaptar a diferentes telas.
  3. Manter a consistência em times diferentes.
  4. Design rápido de protótipos.
  5. Compatibilidade com diversos navegadores.

Desvantagens

  1. Difícil manutenção e leitura de grandes códigos.
  2. Padronização código fonte dos websites que implementam o Bootstrap.
  3. Algumas falhas de lógica.

Instalando/Incluindo o Bootstrap no seu website

Para instalar/incluir o Bootstrap em um projeto, a forma mais prática é importar os arquivos de uma CDN(Content Delivery Network).

Você pode executar os seguintes passos:

  1. <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">

    2. Este código adiciona o link para o arquivo CSS do Bootstrap no seu documento HTML. O atributo href contém o URL do arquivo CSS hospedado no CDN do Bootstrap. Ele é responsável por aplicar os estilos pré-definidos do Bootstrap em seu projeto.

  2. <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js">

    3. Este código inclui o link para o arquivo JavaScript do Bootstrap no seu documento HTML. O atributo src contém o URL do arquivo JavaScript(opcional) hospedado no CDN do Bootstrap. Ele é responsável por fornecer recursos interativos, como modais, abas, carrosséis, entre outros.

Ou

Utilize uma versão local do Bootstrap.

Para isso basta fazer o download do arquivo compactado com os arquivos necessários no site oficial do Bootstrap e arrasta-los para o projeto desejado.

Siga o passo a passo:

  1. Faça o download clicando no botão abaixo

  2. Extraia os arquivos para a pasta do projeto e inclua o path local para os arquivos em vez da URL da CDN.
  3. Utilize o bootstrap normalmente.

Criando a primeira página web com o Bootstrap

  1. Adicione o tipo de documento:

    2. <!DOCTYPE html>

    O bootstrap usa elementos HTML e propriedades que requerem o tipo de documento HTML5.

  2. Garanta que a página será responsiva a dispositivos mobile:

    3. <meta name="viewport" content="width=device-width, initial-scale=1.0">

    O Bootstrap foi feito para ser adequado ao mobile.

  3. Contêineres

    4. O Bootstrap requer elementos contêineres para envolver os conteúdos do site.

    • A classe .container providencia um container com largura fixa.
    • A classe .container-fluid providencia um container com largura máxima, usando a amplitude total da página/viewport.

Sistema de Grid

O sistema de grid do Bootstrap permite criar layouts responsivos e adaptáveis. Ele divide a página em linhas <div class="row"> e colunas <div class="col">. Você pode definir o número de colunas em diferentes tamanhos de tela para criar layouts flexíveis.

Exemplo:


    <div class="container">
        <div class="row">
            <div class="col-sm-6 col-md-4">Coluna 1</div>
            <div class="col-sm-6 col-md-4">Coluna 2</div>
            <div class="col-sm-6 col-md-4">Coluna 3</div>
        </div>
    </div>

Componentes de navegação

O Bootstrap fornece estilos e componentes para criar menus de navegação responsivos e intuitivos. Você pode usar a classe navbar para criar uma barra de navegação e as classes nav e nav-item para criar itens de menu.

Exemplo:


<nav class="navbar navbar-expand-md navbar-light bg-faded">
    <button class="navbar-toggler navbar-toggler-right" type="button" data-toggle="collapse" data-target="#navbarNavAltMarkup" aria-controls="navbarNavAltMarkup" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon"></span>
    </button>
    <a class="navbar-brand" href="#">Navbar</a>
    <div class="collapse navbar-collapse" id="navbarNavAltMarkup">
        <div class="navbar-nav">
            <a class="nav-item nav-link active" href="#">Home <span class="sr-only">(current)</span></a>
            <a class="nav-item nav-link" href="#">Features</a>
            <a class="nav-item nav-link" href="#">Pricing</a>
            <a class="nav-item nav-link disabled" href="#">Disabled</a>
        </div>
    </div>
</nav>

Componentes de botões

O Bootstrap oferece estilos para criar botões atraentes com várias opções, como botões primários, secundários, de sucesso, de perigo, etc. Você pode usar as classes btn e btn-* para aplicar estilos aos botões.

Exemplo:


    <button type="button" class="btn btn-primary">Botão Primário 
    <button type="button" class="btn btn-secondary">Botão Secundário 
    <button type="button" class="btn btn-success">Botão de Sucesso 
    <button type="button" class="btn btn-danger">Botão de Perigo

Conclusão

Essas são apenas algumas das funções básicas do Bootstrap. Existem muitos outros componentes e estilos disponíveis, como modais, formulários, carrosséis, alertas, entre outros. Consulte a documentação oficial do Bootstrap para obter mais informações detalhadas sobre como usar essas funções e explorar outros recursos disponíveis.
